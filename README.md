# Trust Agent Desktop

Production Tauri 2.x desktop application for Trust Agent AI.

## Prerequisites

- Node.js 18+
- Rust toolchain (rustup)
- System dependencies for Tauri: https://v2.tauri.app/start/prerequisites/

## Setup

```bash
npm install
cp .env.example .env
# Fill in API keys in .env
```

## Development

```bash
npm run tauri dev
```

## Production Build

```bash
npm run tauri build
```
